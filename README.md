# Compost-remote
Savonian opinnäytetyö

Prototyyppi kompostin etähallintaan ja monitorointiin

Projektissa käytetty:
- NodeMCU ESP8266
- YL-69
- DS18B20
- 2x DHT11
- Led-valoja
